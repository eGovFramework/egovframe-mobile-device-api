# 기기 정보 (Device Info)

## 개요

기기 정보 기능은 모바일 디바이스의 하드웨어 및 소프트웨어 정보를 조회하고 관리하는 기능입니다. OS 버전, 디바이스 모델, UUID, 네트워크 정보, 연락처 정보 등을 확인할 수 있으며, 수집된 정보를 서버에 저장하고 목록을 조회할 수 있습니다.

## 주요 기능

### 1. 시스템 정보 조회
- 디바이스의 OS 버전, 모델명, 제조사 등의 기본 정보 표시
- 디바이스 고유 UUID 정보 조회
- 프로그램 버전 정보 확인

### 2. 하드웨어 정보
- CPU, 메모리, 저장공간 등의 하드웨어 사양 확인
- RAM 용량 정보 표시

### 3. 네트워크 정보
- 현재 연결된 네트워크 상태 확인
- 네트워크 디바이스 정보 표시

### 4. 앱 정보
- 설치된 앱의 버전 정보 확인
- 프로그램 버전 표시

### 5. 연락처 정보
- 디바이스에 저장된 연락처 개수 조회

### 6. 서버 연동
- 디바이스 정보를 서버에 업로드
- 서버에 저장된 디바이스 정보 목록 조회

## 아키텍처 구조

### Domain Layer
- **Entity**: `DeviceInfo`
  - `os`: 운영체제 정보
  - `uuid`: 디바이스 고유 식별자
  - `pgVer`: 프로그램 버전
  - `ntwrkDeviceInfo`: 네트워크 디바이스 정보
  - `deviceNm`: 디바이스 이름
  - `telno`: 연락처 개수
  - `strgeInfo`: 저장공간 정보

- **Repository**: `DeviceRepository` (인터페이스)
- **UseCase**: `DeviceUseCase`

### Data Layer
- **Service**: `DeviceService`
  - 디바이스 정보 수집
  - 연락처 정보 조회
  - 네트워크 정보 확인

- **Repository Implementation**: `DeviceRepositoryImpl`

### Presentation Layer
- **Screen**: `DeviceInfoPage`
  - 디바이스 정보 표시
  - 정보 업로드 기능
  - 서버 목록 조회 기능

## 의존성
- `device_info_plus: ^12.2.0`: 디바이스 정보 조회
- `package_info_plus: ^8.0.0`: 앱 버전 정보 조회
- `flutter_contacts: ^1.1.8+1`: 연락처 정보 조회

## 주의사항

- 연락처 정보 조회를 위해서는 연락처 접근 권한이 필요합니다
- 네트워크 정보는 현재 연결된 네트워크 상태에 따라 달라질 수 있습니다
- 서버 저장 기능을 사용하려면 웹서버가 실행 중이어야 합니다

## 기술적 특징

- **다양한 정보 수집**: 하드웨어, 소프트웨어, 네트워크, 연락처 등 종합적인 디바이스 정보 제공
- **UUID 생성**: 디바이스 고유 식별자는 `device_info_plus`와 `shared_preferences`를 조합하여 생성되며, 앱 재설치 시에도 유지됩니다
- **권한 관리**: 필요한 권한을 자동으로 요청하고 처리
  - 연락처 정보 조회 시 연락처 접근 권한이 필요하며, 사용자가 거부할 경우 연락처 개수는 표시되지 않습니다
- **플랫폼별 정보**: Android와 iOS에서 제공하는 정보가 다를 수 있으며, 각 플랫폼에 맞는 정보를 수집합니다
- **서버 연동**: 디바이스 정보는 RESTful API를 통해 서버에 저장되며, UUID를 기준으로 관리됩니다
- **Clean Architecture**: 도메인, 데이터, 프레젠테이션 레이어 분리
