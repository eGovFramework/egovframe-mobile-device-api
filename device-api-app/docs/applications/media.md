# 미디어 (Media)

## 개요

미디어 기능은 카메라, 갤러리에서 이미지와 비디오를 선택하고 관리할 수 있는 기능입니다. 선택한 미디어 파일을 로컬에 저장하고, 서버로 업로드하거나 서버에서 다운로드할 수 있습니다. 또한 이미지와 비디오를 앱 내에서 바로 재생할 수 있습니다.

## 주요 기능

### 1. 이미지 선택
- 카메라 또는 갤러리에서 이미지를 선택할 수 있습니다
- 카메라로 직접 촬영 가능
- 갤러리에서 기존 이미지 선택 가능

### 2. 비디오 선택
- 카메라 또는 갤러리에서 비디오를 선택할 수 있습니다
- 카메라로 직접 녹화 가능
- 갤러리에서 기존 비디오 선택 가능

### 3. 미디어 저장
- 선택한 미디어 파일을 로컬에 저장합니다
- 자동으로 파일명 생성 (타임스탬프 기반)

### 4. 미디어 목록
- 저장된 미디어 파일 목록을 확인할 수 있습니다
- 이미지와 비디오를 분리하여 표시
- 그리드 형태로 직관적인 목록 표시

### 5. 미디어 업로드
- 로컬 미디어 파일을 서버로 업로드할 수 있습니다
- 다중 선택 후 일괄 업로드 가능

### 6. 미디어 다운로드
- 서버에서 미디어 파일을 다운로드할 수 있습니다
- 서버 파일 목록에서 다운로드 가능

### 7. 미디어 재생
- 이미지와 비디오를 앱 내에서 바로 재생할 수 있습니다
- 이미지는 전체 화면으로 확대하여 볼 수 있습니다
- 비디오는 재생 컨트롤과 함께 재생됩니다

### 8. 미디어 삭제
- 로컬 및 서버의 미디어 파일을 삭제할 수 있습니다
- 다중 선택 후 일괄 삭제 가능

## 아키텍처 구조

### Domain Layer
- **Entity**: `MediaInfo`
  - `name`: 파일 이름
  - `path`: 파일 경로
  - `size`: 파일 크기
  - `type`: 미디어 타입 (image, video, audio)
  - `lastModified`: 수정일
  - `serverSn`: 서버 일련번호

- **Repository**: `MediaRepository` (인터페이스)
- **UseCase**: `MediaUseCase`

### Data Layer
- **Service**: `MediaService`
  - 이미지/비디오 선택
  - 미디어 파일 저장
  - 서버 업로드/다운로드
  - 미디어 파일 삭제

- **Repository Implementation**: `MediaRepositoryImpl`

### Presentation Layer
- **Screen**: `MediaScreen`
  - 이미지/비디오 탭 분리
  - 미디어 그리드 표시
  - 미디어 선택 모드
  - 이미지 뷰어 (`ImageViewScreen`)
  - 비디오 플레이어 (`VideoPlayerScreen`)

## 의존성

- `image_picker: ^1.0.7`: 이미지/비디오 선택
- `video_player: ^2.8.2`: 비디오 재생
- `permission_handler: ^11.2.0`: 카메라 및 저장소 권한 관리

### 서버 목록 조회

  <image src="images/media_serverList.png" alt="Media 서버 목록 화면">
</div>

서버에 저장된 미디어 파일 목록을 확인할 수 있습니다.

**사용 방법:**

## 주의사항

- 카메라 권한과 저장소 권한이 필요합니다
- 대용량 미디어 파일의 경우 업로드 및 다운로드에 시간이 소요될 수 있습니다
- 서버 저장 기능을 사용하려면 웹서버가 실행 중이어야 합니다
- 비디오 재생 시 배터리 사용량이 증가할 수 있습니다

## 기술적 특징

- **미디어 선택**: 카메라와 갤러리에서 이미지/비디오 선택 지원
  - `image_picker` 패키지를 사용하여 카메라 또는 갤러리에서 미디어를 선택합니다
  - Android와 iOS에서 각각 다른 권한이 필요하며, 자동으로 권한을 요청합니다
- **파일명 생성**: 선택한 미디어는 타임스탬프 기반 파일명으로 저장됩니다
  - 이미지: `{timestamp}.jpg`
  - 비디오: `{timestamp}.mp4`
- **미디어 타입 분류**: 이미지, 비디오, 오디오를 자동으로 분류하여 표시합니다
- **인앱 재생**: 이미지 확대 보기 및 비디오 플레이어 제공
  - 이미지는 `InteractiveViewer`를 사용하여 핀치 제스처로 확대/축소할 수 있습니다
  - 비디오는 `video_player` 패키지를 사용하여 재생하며, 재생 컨트롤을 제공합니다
- **그리드 표시**: 직관적인 미디어 목록 표시
  - 2열 그리드 레이아웃으로 표시되며, 각 미디어의 썸네일과 이름이 표시됩니다
  - 이미지 썸네일은 메모리 최적화를 위해 캐시 크기를 제한합니다
- **다중 선택**: 여러 미디어를 선택하여 일괄 작업 가능
  - 길게 누르기로 선택 모드에 진입하며, 선택된 미디어는 시각적으로 강조됩니다
- **서버 연동**: RESTful API를 통한 미디어 업로드/다운로드
  - multipart/form-data 형식으로 파일을 업로드합니다
  - 서버에 저장된 미디어는 서버 일련번호(serverSn)로 관리됩니다
- **로컬/서버 파일 구분**: 로컬 파일과 서버 파일을 구분하여 표시하며, 서버 파일은 클라우드 아이콘으로 표시됩니다
- **Clean Architecture**: 도메인, 데이터, 프레젠테이션 레이어 분리
