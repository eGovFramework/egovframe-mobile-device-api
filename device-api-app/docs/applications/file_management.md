# 파일 관리 (File Management)

## 개요

파일 관리 기능은 모바일 디바이스의 파일 시스템에 접근하여 파일과 폴더를 생성, 수정, 삭제, 복사, 이동할 수 있는 기능을 제공합니다. 디렉토리 구조를 시각적으로 탐색하고, 다중 파일 선택 및 일괄 작업을 지원합니다.

## 주요 기능

### 1. 폴더 생성

- 새로운 디렉토리를 생성하여 파일을 체계적으로 관리합니다
- 원하는 위치에 폴더를 생성할 수 있습니다

### 2. 파일 생성

- 다양한 형식의 샘플 파일을 생성하여 기능 테스트를 수행합니다
- 텍스트 파일 (.txt), JSON 파일 (.json), 마크다운 파일 (.md) 등 지원

### 3. 파일 탐색

- 디렉토리 구조를 시각적으로 탐색하고 파일 목록을 확인합니다
- 폴더를 탭하여 하위 디렉토리로 이동할 수 있습니다
- 파일 크기, 수정일, 항목 수 등 상세 정보 표시

### 4. 파일 복사

- 선택한 파일이나 폴더를 다른 위치에 복사본을 생성합니다
- 같은 디렉토리 내에서 복사 시 자동으로 "복사본" 이름 생성

### 5. 파일 이동

- 선택한 파일이나 폴더를 다른 디렉토리로 이동합니다
- 같은 디렉토리 내에서 이동 시 이름 변경 기능 제공

### 6. 파일 삭제

- 불필요한 파일이나 폴더를 안전하게 삭제합니다
- 다중 선택 후 일괄 삭제 가능

### 7. 파일 정보

- 파일 크기, 수정일, 항목 수 등 상세 정보를 표시합니다
- 파일 타입에 따른 아이콘 표시

### 8. 다중 선택

- 여러 파일을 동시에 선택하여 일괄 작업을 수행합니다
- 길게 눌러서 선택 모드 진입

## 아키텍처 구조

### Domain Layer

- **Entity**: 파일 시스템 엔티티 (File, Directory)

- **Repository**: `FileManagementRepository` (인터페이스)

### Data Layer

- **Service**: `FileManagementService`
  - 파일 시스템 접근
  - 파일/폴더 생성, 삭제, 복사, 이동
  - 권한 관리

- **Repository Implementation**: `FileManagementRepositoryImpl`

### Presentation Layer

- **Screen**: `FileManagementScreen`
  - 파일 목록 표시
  - 다중 선택 기능
  - 파일 작업 UI

## 의존성

- `file_picker: ^8.0.0+1`: 파일 선택 기능
- `path_provider: ^2.1.2`: 파일 경로 관리
- `permission_handler: ^11.2.0`: 저장소 권한 관리

현재 디렉토리의 파일과 폴더 목록이 표시됩니다. 폴더는 파란색 아이콘으로, 파일은 회색 아이콘으로 표시되며, 각 항목의 상세 정보가 함께 표시됩니다.

- 상단 메뉴(⋮)에서 "새 폴더"를 선택하여 폴더를 생성할 수 있습니다
- 상단 메뉴(⋮)에서 "샘플 파일 생성"을 선택하여 파일을 생성할 수 있습니다

## 주의사항

- 삭제된 파일은 복구할 수 없으므로 신중하게 선택하세요
- 앱의 문서 디렉토리 내에서만 작업이 가능합니다
- 시스템 파일이나 다른 앱의 파일은 접근할 수 없습니다
- Android 11 이상에서는 "모든 파일 관리" 권한이 필요할 수 있습니다

## 기술적 특징

- **파일 시스템 접근**: Dart의 `dart:io` 패키지를 사용한 파일 시스템 조작
  - 앱의 문서 디렉토리(`getApplicationDocumentsDirectory`)에만 접근 가능합니다
  - iOS에서는 샌드박스 환경으로 인해 앱 전용 디렉토리만 접근할 수 있습니다
- **파일 정보 조회**: 파일 크기, 수정일, 항목 수 등의 정보를 비동기로 조회하여 UI 성능을 최적화합니다
- **다중 선택**: 체크박스를 통한 직관적인 다중 파일 선택
  - 길게 누르기(Long Press)로 선택 모드에 진입할 수 있습니다
- **파일 복사/이동 로직**: 같은 디렉토리 내에서 복사 시 자동으로 "복사본" 이름을 생성하며, 중복 시 번호를 추가합니다
- **권한 관리**: 저장소 권한 자동 요청 및 처리
  - Android 11 이상에서는 "모든 파일 관리" 권한이 필요하며, 권한이 없을 경우 안내 다이얼로그를 표시합니다
- **사용자 친화적 UI**: 파일 타입별 아이콘 및 정보 표시
  - 폴더와 파일을 시각적으로 구분하여 표시합니다
- **Clean Architecture**: 도메인, 데이터, 프레젠테이션 레이어 분리
